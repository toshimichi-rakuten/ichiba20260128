# アルコル（ランキング）

商品リスト作成ツール アルコルで利用できるHTMLテンプレートです。モジュールでは、商品リスト部分のみをアルコルのテンプレートとして登録し、実際のページでの見せ方＝レイアウト（何の要素を表示させるか・何カラム表示にするか、スライダーにするか）は、ページ側で設定・変更できるように制作されています。アルコルツール自体について詳しくは、[アルコル（Alcor）](https://confluence.rakuten-it.com/confluence/pages/viewpage.action?pageId=1098887034)（閲覧は社内環境限定）をご確認ください。

## 使い方

1. テンプレートを選んでアルコルを作成する

    <Space />

    アルコルの「ランキング一覧のHTMLを作成する」にある、PCの「HTMLを出力する 」で、アルコル（ランキング）モジュール用のテンプレートである`ecm_alcor_ranking_50.html` を選択して、HTMLを出力してください。PC/SP共通利用が可能です。  
    アルコル側で設定した商品取得件数分の商品順位がページへ表示されます。

    <Space />

2. 表示したい要素・レイアウトに応じたHTMLを選択する

    <Space />

    本ページのレイアウトから、表示したい要素の組み合わせ（価格・商品名・店舗名・レビュー）、使いたいレイアウトのソースコードをコピーし、PC/SPページのHTMLの必要な場所に貼り付けてください。  
    （PC/SP共通利用が可能です）必要に応じて、カラム数を指定するclass部分を書き換えます。  
    カラム数の変更詳細については、グリッドレイアウトを参照。

    <Space />

    <Note type='info'>
      スライダーやスクロール系以外のレイアウトでは、「ランキングをもっと見る」機能があります。「もっと見るボタン付き」のレイアウトを使用すれば使用することができます。
    </Note>

    <Space />

3. アルコルの出力HTMLパスを記載する

    <Space />

    ページのHTMLに追加したレイアウトHTMLの

    <Space />

    <Code content={`
    <!-- Alcor Template -->
   <!--#include virtual="★"-->
   <!-- /Alcor template -->
    `}/>

    <Space />

    includeのパス部分を、出力したHTMLのインクルードパス `/itemlist/20xxxx/****.html` と置き換えてください。  
    最後に、レイアウトが正しく表示されるか確認します。

## 注意事項

    ランキングパーツ（≠ ランキングアルコル）を `event` ページへ実装するときは、免責文を記載することが必須とされているので、ランキングアルコルを入れる場合も、[ランキングパーツ表示に関するルール](https://confluence.rakuten-it.com/confluence/pages/viewpage.action?pageId=2736403238#id-%E3%83%A9%E3%83%B3%E3%82%AD%E3%83%B3%E3%82%B0%E3%83%91%E3%83%BC%E3%83%84-%E3%83%A9%E3%83%B3%E3%82%AD%E3%83%B3%E3%82%B0%E3%83%91%E3%83%BC%E3%83%84%E8%A1%A8%E7%A4%BA%E3%81%AB%E9%96%A2%E3%81%99%E3%82%8B%E3%83%AB%E3%83%BC%E3%83%AB)（閲覧は社内環境限定）ご参考ください。
