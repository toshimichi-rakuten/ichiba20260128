# アルコル（クーポン）

商品リスト作成ツール アルコルで利用できるHTMLテンプレートです。モジュールでは、商品リスト部分のみをアルコルのテンプレートとして登録し、実際のページでの見せ方＝レイアウト（何の要素を表示させるか・何カラム表示にするか）は、ページ側で設定・変更できるように制作されています。アルコルツール自体について詳しくは、[アルコル(Alcor)](https://confluence.rakuten-it.com/confluence/pages/viewpage.action?pageId=1098887034)（閲覧は社内環境限定）をご確認ください。

## 使い方

1. テンプレートを選んでアルコルを作成する

		<Space />

		アルコルの「商品クーポン一覧のHTMLを作成する」にある、PCの「HTMLを出力する 」で、クーポンアルコル用のテンプレートである`ecm_coupon_template.html`を選択して、HTMLを出力してください。PC/SP共通利用が可能です。

		<Space />

2. 下記のフォームを使用してモジュールを設定する

		<Space />

		フォームの設定を変更すると、設定内容に基づいてHTMLコードが自動更新されます。

		<Space />

3. 生成されたHTMLコードをコピーする

		<Space />

		フォームの設定に基づいて、下部にHTMLコードが自動生成されます。  
		生成されたコードをコピーし、ページに貼り付けてください。

		<Space />

4. アルコルの出力HTMLパスを記載する

		<Space />

		出力されたHTMLの

		<Space />

		<Code content={`<!--#include virtual="★"-->`}/>

		<Space />

		includeのパス部分を、出力したHTMLのインクルードパス `/itemlist/xxx/20xxx/***.html` と置き換えてください。  
		最後に、レイアウトが正しく表示されるか確認します。

## 注意事項
本モジュールを利用可能なクーポンはそれぞれ以下のクーポンとなります。  
企画ページに掲載するクーポンが、モジュールで利用可能なクーポンかは各担当者でご確認のほどよろしくお願いいたします。

### 「クーポンのみ」に対応しているクーポン
店舗が発行した店内全品クーポンおよび店舗が発行した商品指定クーポン

### 「商品とクーポン」「商品のみ」に対応しているクーポン
店舗が発行した商品指定クーポン
<Space />
※楽天が発行するクーポンは一律でモジュールの対象外となります。  
※送料無料クーポンはモジュールの対象外となります。  
※店内全品クーポンは商品情報が取得できない為、クーポンのみ利用可能です。  
※上記以外のクーポンはアルコルの機能としては設定可能ですが、モジュールで想定しているケースではございませんので利用は控えてくださいませ。また、出力結果につきましては各担当にて責任を持って設定いただきますようお願いいたします。  
※詳細検索で指定した場合は楽天が発行したクーポンも意図せず表示されてしまう可能性があるため、アルコル設定時にご注意ください。  
※「商品のみ」を選択した場合、クーポン獲得リンク及びクーポン画像は表示されません。別途クーポン獲得ボタンを設置してください。必要に応じて非遷移クーポン獲得ボタンのモジュールもご参照ください。
