## 使い方

- グレーの枠線・白背景が設定されています。（上記プレビューのグレーの背景は含まれません。）変更する際は、CSSユーティリティクラスの追加で対応、または任意クラス名を追加し、その追加任意クラスに対して変更内容を指定してください。
