- テキストサイズの変更は、[ユーティリティクラス](/ecm/docs/css/list/)が利用できます。
- 見出しレベルはh2で提供していますが、ページの構成に合わせて見出しレベル（h1～h6）へ設定を変更してご利用ください。
- 下線なしの場合は、`ecm-content-headline` クラスを削除してください。
