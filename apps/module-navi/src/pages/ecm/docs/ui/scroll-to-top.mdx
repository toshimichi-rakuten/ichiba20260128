# 上に戻る

ページ上にスクロールさせるモジュールです。要素にスクロールしたい場合は [スムーズスクロール](/ecm/docs/js/smooth-scroll/) を使用してください。  
このモジュールはアウトラインファイルにあらかじめ設定された状態で提供されています。
