# パンくずリスト

パンくずリストのモジュールです。**このモジュールはアウトラインファイルにあらかじめ設定された状態で提供されています。**  
スタイルなどは原則変更せず利用してください。

## 使い方

- SP/PC共通で利用することができます。
- `class="d-none md-d-block"` を追加すると、PCのみ表示とすることができます。

<Space />

<Code content={`
<nav aria-label="Breadcrumb" class="d-none md-d-block">
  <ol class="ecm-breadcrumb">〜</ol>
</nav>
`}/>
