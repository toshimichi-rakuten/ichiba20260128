## 使い方

### 各ナビゲーションの幅について

各ナビゲーションの幅はテキスト量に依存します。そのため、幅を固定にしたいなどカスタマイズしたい場合はページごとに個別スタイルを適用させて調整してください。  
また、全体のナビゲーションの幅が画面サイズ（全体のコンテンツ幅）よりも大きい場合は途中で見切れて横スクロールが発生し、小さい場合は中央寄せになります。  

### 表示開始位置の設定

`aria-hidden="true"` を指定  
`data-show-at` の値に表示開始させたい要素のid（#hogehoge）またはclass（.hogehoge）を指定する、または、`data-show-threshold` の値にページ上部からの距離の数値（px単位不要）を指定します。  
※下記サンプルは、`data-show-at` プロパティで実装
