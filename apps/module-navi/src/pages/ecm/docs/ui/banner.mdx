# バナー

ページ内にバナーを設置する場合に利用するモジュールです。 あらかじめレイジーローディングの設定が用意され、各種レイアウトに対応しています。

</Tabs>

## 使い方

- バナーは、各種レイアウト（カラム、スライダー、カルーセルなど）に入るように設計されています。
- `data-lazy-loading=""` に実際表示させる画像パスを記載してください。
- `src="https://r.r10s.jp/com/img/home/t.gif?resize=400:400"` の `?resize=400:400` 部分は実際の画像サイズ（横:縦）へ編集してください。
- HTMLを必要なバナーの数だけコピーして、レイアウトと組み合わせて下さい。
