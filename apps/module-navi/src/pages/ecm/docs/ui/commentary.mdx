# 下層ページ一覧

企画ページに複数の下層ページがある場合に、下層ページコンテンツの紹介を一覧掲載する際などに利用できるモジュールです。

## 使い方

`ecm-commentary` 要素をページで必要な数だけ複製をしてください。  
レイアウトはページで見せたいレイアウトに合わせて、グリッドシステムを利用して変更できます。各行のカラム数は、SP表示 `d-grid-col-*`、PC表示 `md-d-grid-col-*` のクラスの `*` を `1〜12` までの値にして変更できます。  
テキストサイズの変更は、ユーティリティクラスが利用できます。
