## 使い方

必要なパラメーターを `<form>` タグの中へ `<input type="hidden"～` の記載で追加してください。

- ジャンル ID を入れる場合；`<input type="hidden" name="g" value="★ジャンルIDを入れる★">`
- タグ ID を入れる場合；`<input type="hidden" name="tg" value="★タグIDを入れる★">`
- マーケティングキーワードを入れる場合；`<input type="hidden" name="mk" value="★JINGIで発行されたキーワードを入れる★">`
- l-id を入れる場合；`<input type="hidden" name="l-id" value="★付けたいl-idの値を入れる★">`

### 注意事項

- 必要なパラメーターは必ずディレクターに確認をして設定してください。
- セレクトオプションを利用する場合、セレクトオプションの `name` の値は、`sitem` 以外（`g、tg` など）でご利用ください。
- ひとつのフォーム内で複数の（`name="sitem"`）を設定することはできません。
- パラメーターについて詳細は[マニュアル](https://confluence.rakuten-it.com/confluence/display/JISRCH/3.URL+Spec)（閲覧は社内環境限定）参照
