# ハンバーガーメニュー

上部に固定表示されるハンバーガーメニューつきのナビゲーションです。SP用です。

<Space />

<Tabs keepMounted={false} defaultValue='sp-above' styles={{panel: { paddingTop: '24px' }}}>
  <Tabs.List>
    <Tabs.Tab value='sp-above'>イメージ</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value='sp-above'>
    <Html device='sp' content={aboveDemoHtml}/>
  </Tabs.Panel>
</Tabs>

## 使い方

このハンバーガーメニューのタグ（HTML）が記述された位置にスクロールが到達したときに、表示開始され、ページ上部にフローティングします。  
各パターンでソースコード編集方法ご確認ください。

1. 横並びメニューのパターンを決める
    - 割り付け・テキストのみ
    - 割り付け・アイコンあり
    - 割り付け・テキストボタンタイプ
    - スクロール・テキストのみ
    - スクロール・アイコンあり
    - オプション    
2. ハンバーガーメニューの開き方（上から開く／右から開く）を決める
    - 上から開く
    - 右から開く

<Space />

<Note type='info'>
  表示開始位置などフローティングについては[フローティング](/ecm/docs/js/floating/)を参照  
  上から開く設定詳細については[シート](/ecm/docs/ui/sheet/)を参照  
  右から開く設定詳細については[サイドバー](/ecm/docs/ui/sidebar/)を参照
</Note>

### 表示開始位置の設定

`aria-hidden="true"` を指定  
`data-show-at` の値に表示開始させたい要素のid（#hogehoge）またはclass（.hogehoge）を指定する、または、`data-show-threshold` の値にページ上部からの距離の数値（px単位不要）を指定します。  
※上記ソースコードは、`data-show-threshold` プロパティで実装

<Space />

<Note type='info'>
  詳細設定については[フローティング](/ecm/docs/js/floating/)を参照
</Note>

### メニュー内の設定

ハンバーガーメニューを開いたときは、下記div要素内に、画像など任意のコンテンツを追加することができます。  

<Space />

**上から開く場合**  
`<div class="ecm-sheet-content-body">〜</div>`

<Space />

**右から開く場合**  
`<div class="ecm-sidebar-content">〜</div>`

<Space />

<Note type='info'>
  既存のメニュー内リンクリストの詳細設定については[アコーディオン](/ecm/docs/ui/accordion/)を参照
</Note>
