## 使い方

フォントサイズに対して適切な余白を指定されています。フォントサイズは変更せず利用することを推奨します。

<Space />

<Note type='info'>
  横並び余白について変更する場合は[CSSユーティリティ](/ecm/docs/css/)の `d-flex-gap-{4-80}` で指定してください
</Note>
