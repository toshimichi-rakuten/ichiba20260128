# コンテナー

コンテンツエリアを定義するモジュールです。このモジュールはアウトラインファイルにあらかじめ設定された状態で提供されています。  
各モジュールは、このコンテナーの中に配置して利用します。

## SP/PC1カラムアウトライン共通

<Tabs keepMounted={false} defaultValue='sp' styles={{panel: { paddingTop: '24px' }}}>
  <Tabs.List>
    <Tabs.Tab value='sp'>SP</Tabs.Tab>
    <Tabs.Tab value='lg'>PC</Tabs.Tab>
    <Tabs.Tab value='code'>コード</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value='sp'>
    <Html contentPadding="0px" content={defaultDemoHtml}/>
  </Tabs.Panel>

  <Tabs.Panel value='lg'>
    <Html contentPadding="0px" device='lg' content={defaultDemoHtml}/>
  </Tabs.Panel>
  
  <Tabs.Panel value='code'>
    <Code content={defaultHtml}/>
  </Tabs.Panel>
</Tabs>

<Space />

- コンテナー（`ecm-container`要素）のスタイル指定は、左右パディング `16px`、左右マージン `auto`、横幅 `100%` が設定されています。
- PC1カラムアウトラインでは、左右パディング `32`px、横幅 `1024px`（`min-width:1024px; max-width:1024px;`）コンテンツ内容幅 960px が適用されます。
