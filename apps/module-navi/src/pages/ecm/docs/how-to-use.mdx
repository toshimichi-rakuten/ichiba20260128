# 使い方

本サイト・モジュールの使い方を掲載しています。

## Module Naviの使い方

**推奨環境**  
Module Navi（本リファレンスサイト）の利用推奨環境は、Google Chrome、Firefox、Microsoft Edge（各最新版）です。

<Space />

**モジュールの探し方**  
左ナビゲーションの検索窓は、原稿タイプID・フリーワードで検索できます。  
原稿タイプ用モジュールを探す場合は、[AdFinderページ](/ecm/docs/ad-finder/)で、要件を選択し絞り込み探すことができます。  
その他UIパーツについては、検索窓のフリーワード検索のほか、[一覧ページ](/ecm/docs/ui/all/)でプレビューから探すこともできます。

<Space />

**モジュールパーツ詳細**  
各パーツのページには、動作プレビュー・ソースコード・使い方・動作設定方法が掲載されています。  
動作プレビュー内でのリンク遷移は正しく動かない場合があります。ブラウザの戻るボタン、またはページを再読み込みでプレビュー表示に戻すことができます。

## モジュールの使い方

1. アウトラインファイルのダウンロード  
   本サイト左ナビゲーションの最新アウトラインファイルダウンロードボタンから、一式ダウンロードできます。  
   PCアウトラインファイルは、ページの基本レイアウトによって種類がわかれていますので、  
   ダウンロードした一式ファイルのなかから、使用したいレイアウトのファイルを使用してください。

<Space />

2. アウトラインファイルのincパスの編集  
   企画・グループごとに、共通CSS/JSの読み込みパスが指定されています。  
   includeファイルの **★★groupName★★/★★groupName★★_★★groupNumber★★** を、制作するページの該当するグループのものに変更します。[incパス指定表（閲覧は社内環境限定）](https://confluence.rakuten-it.com/confluence/pages/viewpage.action?pageId=4779738706)

<Space />

3. モジュールパーツの配置  
   コンテンツ内で使いたいパーツを探し、[copy]ボタンでHTMLソースをコピーし、アウトラインファイルへ貼り付けます。  
   パーツの動作設定については、ページ内の説明を参考に適宜設定を調整します。

<Space />

4. テキストや、画像のパス、広告タグ部分などを、適宜編集します。  
   ページ個別のCSSファイル、またはモジュールのCSSユーティリティを利用して、デザインを調整します。  
   モジュールパーツでは、基本となるレイアウトやスタイルのみ設定してあり、そのまま利用いただけますが、  
   色・背景などのデザインを反映したい場合は、各ページ固有のCSSファイルで指定してください。
