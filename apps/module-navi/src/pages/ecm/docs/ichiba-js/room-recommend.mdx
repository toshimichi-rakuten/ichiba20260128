## 使い方

1. 商品リスト作成ツールAlcorトップより ROOM一覧のHTMLを作成するを選択ください。（使用希望の際は、ROOM担当者までご連絡ください。）
2. 出力フォーマットをHTMLとJSONで選ぶことができますが、どちらを選択いただいても問題ありません。
3. 上記ソースコードをコピー・貼り付け後、`data-json-url=""`の値を アルコルから出力されたHTMLまたはJSONのパスへ置き換えてください。
