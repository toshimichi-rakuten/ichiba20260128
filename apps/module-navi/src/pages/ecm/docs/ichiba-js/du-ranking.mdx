# ランキングパーツ

ランキングをパーツ形式で表示することができます。ランキングパーツには様々なランキングを設定をすることが可能です。  
利用にあたっては、制作依頼を申請しソースコードを受け取ることが必要です。[ランキングパーツ](https://confluence.rakuten-it.com/confluence/pages/viewpage.action?pageId=2736403238)（閲覧は社内環境限定）ページを参照し申請してください。

<Space />

本ページはパーツのレイアウト動作サンプルを表示します。このソースコードは個別ページでは利用できません。

<Space />

<Note type='warn'>
  スライダー機能はパーツ独自のもので、モジュールのスライダー機能の設定方法とは異なり、本ページでは正しく動作しません。実際のページで確認してください。
</Note>
