# オプショナルスタイル

## Rakuten Condensed

<Html
  full={true}
  device={'lg'}
  rakutenCondensed={true}
  content={rakutenCondensedHtml}
/>

### 使い方 [condensed]

#### 1. CSSの読み込み [condensed]

HTML内で、適用するCSSファイルを下記の通り記述する。
<Space/>
<Code content={`
<link
  rel="stylesheet"
  href="https://r.r10s.jp/com/js/c/rakuten_sans/rakuten_condensed/1.0/rakuten_condensed-1.0.0.min.css?v=202512221523"
/>
`} />

#### 2. フォントの適用 [condensed]

<Code content={`
<!-- Light (300) -->
<div class="font-rakuten-condensed font-weight-300">
  Light Weight Text
</div>

<!-- Regular (400) -->
<div class="font-rakuten-condensed font-weight-400">
  Regular Weight Text
</div>

<!-- Semi-Bold (600) -->
<div class="font-rakuten-condensed font-weight-600">
  Semi-Bold Weight Text
</div>

<!-- Bold (700) -->
<div class="font-rakuten-condensed font-weight-700">
  Bold Weight Text
</div>

<!-- Black (900) -->
<div class="font-rakuten-condensed font-weight-900">
  Black Weight Text
</div>

<!-- 変数フォントのため、任意の数値も指定可能 -->
<div class="font-rakuten-condensed" style="font-weight: 442;">
  Custom Weight
</div>
`} />

<link
  rel="stylesheet"

/>
`} />

##### 600 (SemiBold)

<Code content={`
<link
  rel="stylesheet"
  href="https://r.r10s.jp/com/js/c/rakuten_sans/rakuten_sans_jp2_600/1.0/rakuten_sans_jp2_600-1.0.0.min.css?v=202512221523"

##### 800 (Heavy)

<Code content={`
<link
  rel="stylesheet"

/>
`} />

#### 2. フォントの適用 [jp2]

<Note>
  このフォントはウェイトごとに分割されているため、`font-rakuten-sans-jp2 font-weight-800` のような組み合わせは機能しません。ウェイトごとのクラス（例：`font-rakuten-sans-jp2-800`）を使用してください。
</Note>

<Space />

<Code content={`
<!-- Regular (500) -->
<div class="font-rakuten-sans-jp2-500">
  楽天市場へようこそ
</div>

<!-- Bold (800) -->
<div class="font-rakuten-sans-jp2-800">
  重要なお知らせ
</div>
`} />
