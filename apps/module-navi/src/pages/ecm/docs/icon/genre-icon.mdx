# ジャンル アイコン

楽天市場で利用される、ジャンルのアイコンをまとめたモジュールです。

<Space />

<IconGrid type="genre" />

## 使い方

1. 一覧から利用するアイコンを探し、クリックするとソースコードをコピーできます。

    <Space />

    HTMLに貼り付け、必要に応じて各ページ任意のclass名を追加します。

    <Space />

    <Code content={`
    <ul class="sample">
     <li class="sample-list"><i aria-hidden="true" class="ecm-genre-icon-accessories sample-list-icon"></i>アクセサリー</li>
     <li class="sample-list"><i aria-hidden="true" class="ecm-genre-icon-sports sample-list-icon"></i>スポーツ</li>
   </ul>
    `}/>

    <Space />

2. ページ側CSSで大きさ・色を調整してください。Web Fontのため、`font-size/color` で変更してください。

    <Space />

    <Code language='css' content={`
    .sample-list-icon {
     /* アイコンの色を設定 */
     color: #bf0000;
     /* アイコンの大きさを設定 */
     font-size: 24px;
   }
    `}/>

### 注意事項

- デザインレギュレーションの変更に伴って、モジュールの更新に関わらず表示されるアイコンが変わる可能性があります。ご了承下さい。
- **それぞれのアイコンには利用目的があります。必ずディレクター・デザイナーの指示に基づいてご利用ください。**
- モジュールのクラス名には直接スタイルを設定せず、ページ側で追加したクラス名にスタイルを設定してください。
