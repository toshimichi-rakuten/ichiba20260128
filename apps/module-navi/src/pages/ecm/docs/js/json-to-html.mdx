# JSONtoHTML

**※コーダー/エンジニア向け**    
<Space />

JSONデーターをHTMLに表示するモジュールです。

## 使い方

### JSONデーター式

<Tabs keepMounted={false} defaultValue='sp' styles={{panel: { paddingTop: '24px' }}}>
  <Tabs.List>
    <Tabs.Tab value='sp'>SP/PC</Tabs.Tab>
    <Tabs.Tab value='code'>コード</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value='sp'>
    <Html content={jsonStringHtml} />
  </Tabs.Panel>
  
  <Tabs.Panel value='code'>
    <Code content={jsonStringHtml}/>
  </Tabs.Panel>
</Tabs>

### HTMLデーター式

<Tabs keepMounted={false} defaultValue='sp' styles={{panel: { paddingTop: '24px' }}}>
  <Tabs.List>
    <Tabs.Tab value='sp'>SP/PC</Tabs.Tab>
    <Tabs.Tab value='code'>コード</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value='sp'>
    <Html content={defaultHtml} />
  </Tabs.Panel>
  
  <Tabs.Panel value='code'>
    <Code content={defaultHtml}/>
  </Tabs.Panel>
</Tabs>

## イベント

<Code language='js' content={`
// Set data
j2h.dispatchEvent(new CustomEvent('ecmJsonToHtmlSetData', { detail: data }))
// Load templates
j2h.dispatchEvent(new Event('ecmJsonToHtmlLoadTemplates'))
// Render
j2h.dispatchEvent(new Event('ecmJsonToHtmlRender'))
`}/>
