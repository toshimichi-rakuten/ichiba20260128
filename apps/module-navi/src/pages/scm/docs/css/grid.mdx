# グリッドレイアウト

グリッドレイアウトは、カラム配置でレイアウトできるモジュールです。使い方の基本は [CSSユーティリティクラス](/ecm/docs/css/) と同様です。

## 主な機能・特徴

- 12カラムベースのレスポンシブグリッドシステムです。
- PC用のクラス（接頭辞 `md-`）を使用することで、1つのソースコードでPCとSPでレイアウトを変更することができます。
- 各グリッドアイテムは等しい幅と高さを持つことになります。

## 使い方

各行のカラム数と、各グリッドアイテム間（行下/列間）のスペースをクラス名で指定します。  
カラムで並べる要素の外側（親）要素に、以下の各クラスを設定します。

<Space />

> `d-grid`：グリッドレイアウトを適用  
> `d-grid-col-★`：★カラムで表示（★＝`1～12` の整数）  
> `d-grid-gap-☆`：要素の間、下の余白が☆pxになる（☆＝`4～80` の `4` の倍数）

<Space />

例）3カラム、要素の余白（下、間）16px

### デバイスごとに指定したい場合

デバイスごとにレイアウトを表示分けする場合は、[CSSユーティリティクラス](/ecm/docs/css/)と同様、PC用に接頭辞を付けたクラス名を追加します。  
以下の各クラスを設定します。（接頭辞が無いものはSP/PC反映、接頭辞があるものでPC用に上書き反映されます。）

<Space />

> `d-grid`：グリッドレイアウトを適用  
> `d-grid-col-★`：★カラムで表示（★＝`1～12` の整数）  
> `d-grid-gap-☆`：要素の間・下の余白が☆pxになる（☆＝`4～80` の `4` の倍数）  
> `md-d-grid-col-●`：PC幅のときは●カラムで表示（●＝`1～12` の整数）  
> `md-d-grid-gap-○`：PC幅のときは要素の間・下の余白が○pxになる（○＝`4～80` の `4` の倍数）

<Space />

例）SP：3カラム、要素の余白（下・間）16px、PC：5カラム、要素の余白（下・間）24px

### 余白の指定を上下と左右で分けたい場合

左右間の余白は `d-grid-gap-x-` 上下の余白は `d-grid-gap-y-` で分けて指定することができます。

<Space />

> `d-grid`：グリッドレイアウトを適用  
> `d-grid-col-★`：★カラムで表示（★＝`1～12` の整数）  
> `d-grid-gap-x-☆`：要素の間の余白が☆pxになる（☆＝`4～80` の `4` の倍数）  
> `d-grid-gap-y-■`：要素の下の余白が■pxになる（■＝`4～80` の `4` の倍数）  
> `md-d-grid-col-●`：PC幅のときは●カラムで表示（●＝`1～12` の整数）  
> `md-d-grid-gap-x-○`：PC幅のときは要素の間の余白が○pxになる（○＝`4～80` の `4` の倍数）  
> `md-d-grid-gap-y-□`：PC幅のときは要素の下の余白が□pxになる（□＝`4～80` の `4` の倍数）

<Space />

例）SP：3カラム、要素間の余白16px、要素下の余白24px、PC：5カラム、要素間の余白24px、要素下の余白32px
