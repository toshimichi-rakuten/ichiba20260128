# AIサーチ

## 使い方

### 1. JSの読み込み

JSをページに読み込んでください  

<Code content={`
<script
  src="${getScmJsHref()}"
  type="text/javascript"
></script>
`}/>

<Space />

位置は`ページJavaScript`の方に入力してください  

<Code copy={false} content={`
  <!-- ==================== ページJavaScript ==================== -->
  <!-- ページで作成するJavaScriptはここに追加してください。-->
  <script
    src="${getScmJsHref()}"
    type="text/javascript"
  ></script>
  <!-- ==================== /ページJavaScript ==================== -->
</body>
`}/>

<Space />

<Note type='info'>
  SCM JSは、AISearchモジュールを使用するページでのみ必要です。
</Note>

<Space />

### 2. HTMLをページに追加

<Tabs

### 3. 設定を調整

企画によって、`<style>`と`<script data-settings>`を調整してください

<Space />

<Code copy={false} language="html" content={`
<style>
  :root {
    (中身を調整)
  }
</style>

<script data-settings="" type="application/json">
  (中身を調整)
</script>
`}/>
