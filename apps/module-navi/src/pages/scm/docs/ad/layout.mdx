# 広告のレイアウトについて

広告のモジュールのレイアウト（カラム数・スペース）は、親要素を追加して調整する必要があります。  
詳しくは[グリッド](/scm/docs/css/grid/)を確認してください。  

例）`d-grid-col-4`こちらのクラスで**4**列でセットされています。コードを確認してください。

    <Tabs.Tab value='demo'>デモ</Tabs.Tab>
    <Tabs.Tab value='code'>コード</Tabs.Tab>
  </Tabs.List>

  <Tabs.Panel value='demo'>
    <Html scm={true} device='md' full={true} content={`
      <div class="d-grid d-grid-gap-16 d-grid-col-4">
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
        ${ad100183doorsoloHtml}
      </div>
    `}/>
  </Tabs.Panel>
  
  <Tabs.Panel value='code'>
